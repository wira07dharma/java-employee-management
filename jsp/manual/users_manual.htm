2.6.2          Attendance

Attendance module is used to manage employee attendance. The attendance module consists several parts, i.e.: Working Schedule and Presence.

 

2.6.2.1         Working Schedule

This part manages employees working schedule. Click Employee > Attendance > Working Schedule to open the employees working schedule searching page.

  • Requirements:
    • Period taken from Master Data > Schedule > Period
    • Schedule Symbol taken from Master Data > Schedule > Symbol

 

  • Processes:
    • Create new working schedule

You can create a new working schedule for an employee by several ways: creating from scratch, copying from existing employees working schedule, or import from Excel file.

      1. From scratch:

1.       Click Employee > Attendance > Working Schedule

2.       Click Add New Schedule

3.       Select period from Period list the working schedule calendar will be displayed

4.       Select one employee using searching form

5.       Fill the working schedule calendar by choosing Schedule Symbol

6.       Save the working schedule

 

      1. Copy from existing employees working schedule

1.       Click Employee > Attendance > Working Schedule

2.       Click Search for Schedule

3.       Click one employees working schedule you wish to copy the employees working schedule calendar will be displayed

4.       Click the Create new Working Schedule based on this working schedule to copy the displayed working schedule

5.       Select one employee using searching form

6.       Fill the working schedule calendar by choosing Schedule Symbol

7.       Save the working schedule

Figure 13 Creating new working schedule from copy

 

      1. Import from Excel file

1.       Click Employee > Attendance > Working Schedule

2.       Click Import from Excel button

3.       Click Browse button

4.       Choose the Excel file then click Open button

5.       Click Submit button to upload file to server

6.       If the format of Excel file is correct, you should get your screen similar to the following result.

 

Figure 14 Result of Excel working schedule import

 

If there is no working schedule displayed, please check the Excel working schedule format, it must be the same with the original (sample) Excel working schedule file as below:

 

Figure 15 Original (sample) of Working Schedule in MS Excel

7.       Select the period from Period of Working Schedule list

8.       Click Save Working Schedule to save the imported working schedule to database

 

    • Changing working schedule
      1. Click Employee > Attendance > Working Schedule
      2. Click Search for Schedule
      3. Click one employees working schedule you wish to update, then the employees working schedule calendar will be displayed
      4. Update the working schedule calendar by choosing Schedule Symbol
      5. Save the working schedule

 

2.6.2.2         Manual Registration

This part manages employees presence

    • Create new manual registration:
      1. Click Employee > Attendance > Manual Registration
      2. Click Add New Presence
      3. Select one employee using searching form
      4. Fill the Presence Date & Time
      5. Choose one of the presence status from the status list:

         In start working

         Out Home finish working

         Out On duty out of office (on duty)

         In Lunch coming back to the office (lunch)

         In Break coming back to the office (personal)

         In Callback coming back to the office (duty)

 

    • Edit the existing presence record:

1.       Click Employee > Attendance > Manual Registration

2.       Fill the searching form criteria if necessary

3.       Click Search for Presence

4.       Click one of employee presence you wish to edit

5.       Fill the Presence Date & Time

6.       Choose one of the presence status from the status list

3.1      What Should I Do?

 

3.1.1          Accessing Harisma Application

You can access the Harisma application in several ways:

  1. Clicking the Harisma application icon on the desktop
  2. Open an Internet Explorer program and supply the following address in the address bar: http://192.0.0.10:8080/harisma/

Then you will see the login screen of Harisma application or message below:

 

Figure 37 Error message if the application server is not run

The message above tells that the Harisma application server is not run due to several causes:

  1. The server computer is off
  2. The server computer is on but the Harisma application is not run
  3. The server computer and the Harisma application is run but something prevent your computer to connect to Harisma server, i.e. network problem

Please contact the system administrator and or EDP staff to solve the trouble

 

Note for System Administrator

To solve this trouble, please refer to the following step:

  1. Make sure the server is on
  2. Make sure whether the Harisma application server is run. You will see the Tomcat 3.2 window below if the Harisma application server is run.

 

Figure 38 Harisma application server

3.       Check the connection between server and client.

 

3.1.2          Login Screen

Before you can use the Harisma application, you have to supply the Login Name and Password in the login screen. If you type wrong Login Name or Password, the system will give you message, for example The system cannot login you or Login ID or Password is invalid.

Figure 39 Login message: "The system cannot login you"

 

Figure 40 Login message: "Login ID or Password is invalid"

Please supply the correct Login Name and Password or ask the System Administrator to solve this trouble.
